Quick Start

1

Install GitHub Copilot CLI

Make sure you have GitHub Copilot CLI installed and authenticated.

npm install -g @githubnext/github-copilot-cli && github-copilot-cli auth
2

Run openRAPPter

Start chatting immediately with npx (no install needed).

npx openrappter
3

Or Install Globally

For faster startup, install globally.

npm install -g openrappter
4

Start Using!

Run tasks, chat, or let it evolve in the background.

openrappter --task "explain this codebase"
