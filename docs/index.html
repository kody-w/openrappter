Features

Everything you need. Nothing you don't.

A minimal, powerful framework for developers who want production-grade AI agents without the complexity or the bill.

📄

Single File Agents

Documentation, contract, and deterministic code all live in one file. No YAML. No config drift. No magic parsing. The file IS the agent.

🔐

Zero API Keys

Powered by the GitHub Copilot SDK. No separate keys for OpenAI, Anthropic, or anyone else. One less secret to manage, one fewer bill to pay.

🏠

Local-First

Your memory, configs, and conversation history live in ~/.openrappter on your own machine. Data stays local even though Copilot powers the AI.

🧠

Persistent Memory

Hybrid search over embedded memory chunks. Remembers facts, preferences, and context across every session. Gets smarter the more you use it.

⚡

5 LLM Providers

Anthropic, OpenAI, GitHub Copilot, Ollama, and Azure OpenAI all supported out of the box. Switch with a config change, no code edits required.

📶

15+ Channels

CLI, Slack, Discord, Telegram, Signal, iMessage, Google Chat, Teams, WhatsApp, Matrix, and more. Your agents meet you where you already are.

🔭

WebSocket Gateway

JSON-RPC 2.0 protocol with streaming agent responses. Event system for agent, chat, channel, cron, and presence events. Real-time everything.

🔌

Plugin System

ClawHub skills are SKILL.md files that wrap into full BasicAgent instances. Search, install, and share in one command.

🔄

Dual Runtime (TS + Python)

Parallel implementations in TypeScript and Python that mirror the same agent architecture. Write once in either language, deploy anywhere.

Architecture

How it works

Every agent call passes through a clean execution pipeline. Data sloshing enriches context before your agent logic ever runs — temporal signals, memory echoes, behavioral hints, and upstream chain data all flow in automatically. You write perform(); the framework handles everything else.

execute(kwargs) │ ├── slosh(query) ── gather context: temporal, memory echoes, hints, priors │ └─ synthesize Orientation { confidence, approach, hints } │ ├── merge upstream_slush ── chain data from previous agent (if any) │ ├── perform(kwargs) ── your agent logic runs here │ └── extract data_slush ── pass enriched output to next agent in chain

The TypeScript and Python runtimes share the same agent contract: BasicAgent, ShellAgent, MemoryAgent, and the ClawHub skill wrapper are all implemented in parallel. Write your agent in either language — the architecture is identical.
