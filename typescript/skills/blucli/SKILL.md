# BluCLI

Control Bluetooth devices from the command line.

## List Devices

```bash
blucli list
```

## Connect a Device

```bash
blucli connect "Device Name"
```

## Disconnect

```bash
blucli disconnect "Device Name"
```

## Check Status

```bash
blucli status
```
